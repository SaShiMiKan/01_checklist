# 01_checklist
第1回の講義課題

## 課題の内容

この template リポジトリを用いて新しく自分専用のリポジトリを作成し、自分の環境にクローンしてローカルリポジトリを作成し、以下のチェックリストを書き換えて（チェックを入れて）、コミット・プッシュしなさい。

チェックリストの書き換えとコミット・プッシュができれば、git の最低限の使い方を理解していることの証明となります。

---

### Git の基本操作

- [ ] このリポジトリをローカルにクローン（clone）した
- [ ] クローンしたリポジトリをエディタまたはコマンドラインで開いた
- [ ] このファイル（README.md）を編集した
- [ ] 変更のステージング（git add）を理解した
- [ ] コミット（git commit）を理解した
- [ ] プッシュ（git push）を理解した
